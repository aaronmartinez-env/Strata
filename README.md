# AERIS

Atmospheric Interpretation System for Valencia Air Quality

AERIS is an environmental analysis pipeline designed to investigate
relationships between measured air pollution, meteorological conditions,
and perceived air quality in Mediterranean urban systems.

## Research Objective

How accurately do public air-quality representations reflect
actual atmospheric conditions in Valencia during normal
and dust-intrusion events?

## System Components

- Scientific AQI model
- Calima detection
- Air-mass classification
- Source attribution
- Atmospheric Complexity Index
- Spatial interpolation
- Wind vector analysis
- 3D urban atmospheric visualization
